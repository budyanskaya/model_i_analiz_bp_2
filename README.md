# Моделирование и анализ бизнес процессов
## Задача 2 
*Сценарий:* «Обработка возврата товара в интернет-магазине»
Описание: Клиент подаёт заявку на возврат → оператор проверяет условия → приём товара на складе → проверка состояния → решение о возврате средств или отказе → уведомление клиента.
Задание. Моделирование в StormBPMN (45 мин)
1. Создайте новый процесс «Возврат товара» в реестре процессов.
2. Воспроизведите ту же логику, что и в Draw.io, с сохранением всех элементов BPMN 2.0.
3. Используйте функцию привязки ролей к задачам (настройте справочник ролей: оператор, кладовщик, бухгалтер).
4. Добавьте атрибуты к задачам: длительность (в часах), исполнитель.
5. Опубликуйте версию диаграммы и получите ссылку для просмотра.
